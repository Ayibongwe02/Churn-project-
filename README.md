# 🚨 CHURN SIGNAL - Telco Customer Churn Prediction Dashboard

An interactive **Streamlit** web application for predicting and analyzing customer churn in the telecommunications industry.



## ✨ Features

- **Risk Distribution** – Visual overview of churn risk across the entire customer cohort
- **High-Risk Leaderboard** – Top customers ranked by predicted churn probability
- **Segment Analysis** – Churn rates by Contract Type and Internet Service
- **Revenue at Risk** – Track potential monthly revenue loss by risk band
- **Live SQL Queries** – Run and explore the underlying analytics directly in the app
- **Responsive & User-Friendly** – Clean, professional interface designed for business users

## 🛠️ Tech Stack

- **Frontend**: Streamlit
- **Data Processing**: Pandas
- **Visualization**: Plotly
- **Database**: SQLite (in-memory)
- **Backend**: Python + SQL

## 📊 Data

- Synthetic Telco-style dataset modeled on real-world churn patterns
- Includes customer demographics, service details, contract info, and predicted risk scores
